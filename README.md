# NASA Space Apps Challenge Project | InfotronX_2

This repository contains the source code for our **NASA Space Apps Challenge** project, developed by **InfotronX_2**.
I am publishing it on my GitHub as part of my portfolio, since I was the main contributor to the project and worked on most of the implementation.
The project also reflects team collaboration during the season.

🏆 **Global Nominee** — our team was internationally recognized as a **Global Nominee** in the NASA Space Apps Challenge.

## About the Project

This project is an interactive web application built around space exploration, planets, and visual presentation.

It was designed to offer a dynamic and engaging experience through custom pages, visual effects, and interactive elements implemented with front-end web technologies.

## Main Highlights

- Space-themed interactive website
- Dedicated pages for multiple planets and celestial elements
- Visual assets and 3D components
- Custom front-end implementation
- Clean browser-based access through an online deployment

## Technologies Used

- HTML
- CSS
- JavaScript
